# 思飞无人机 nuc

终端1
sudo chmod 777 /dev/ttyACM0 
roslaunch mavros px4.launch 

注意px4也要修改ip，不然链接不上qgc

开启px4

终端2

roslaunch livox_ros_driver2 msg_MID360.launch

启动雷达驱动

终端3
roslaunch fast_lio mapping_mid360.launch

启动fastlio建图

终端4

source ~/racer/devel/setup.bash

rosrun cxr_egoctrl_v1 offb_node

转换程序，发布传感器位置vision_pose

终端5

cd ~/racer/src/scripts

python3 ego_transfer.py

发布camera——pose

终端6

source ~/racer/devel/setup.bash

rosrun cxr_egoctrl_v1 trans.py

这个是偏置转换程序

终端7

source ~/racer/devel/setup.bash

rosrun cxr_egoctrl_v1 cxr_egoctrl_v2

程控，最后开！！！！

终端8

source ~/racer/devel/setup.bash

roslaunch exploration_manager rviz.launch

rviz传图界面

终端9

source ~/swarm_ros_bridge_ws/devel/setup.bash

roslaunch swarm_ros_bridge bridge_drone1.launch

双机通信程序

终端10

source ~/racer/devel/setup.bash

roslaunch exploration_manager swarm_exploration_realworld.launch 2>&1 | tee "/home/nuc/文档/swarm_$(date +%Y%m%d_%H%M%S).log"

主探索程序，飞机稳定后开启，注意停止，这样可以保存log！！！！！！，我这个nuc保存在文档下面
